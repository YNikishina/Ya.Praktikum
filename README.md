# Мои учебные проекты

В данном репозитории размещены примеры учебных проектов, которые я самостоятельно выполняла в процессе обучения в Яндекс Практикуме. Большая часть проектов была принята ревьюерами с первого раза без необходиомости доработки.

В результате анализа сделаны выводы о корректности проведения теста (соответствие техзаданию) и оценены результаты теста.
